# Editor

Set a favorite code editor once, then open any site's folder in it with a click.

Open **Settings → Editor**.

The dropdown **auto‑detects editors installed** on your Mac, including:

- Visual Studio Code
- Cursor
- Windsurf
- Zed
- Sublime Text
- PhpStorm
- Nova
- VSCodium
- BBEdit

Pick one and it's saved immediately.

### Not in the list?

Choose **Other…** and type either:

- an **app name** (used with `open -a`, e.g. `Visual Studio Code`), or
- a **command** (e.g. `code`, `cursor`, `subl`).

Then click **Save**.

## Open a site in the editor

From a site's **⋯ menu → Open in editor**.

This opens the site's **`site/` folder** — your editable content, themes, and `config.json` — rather than the whole FrontPress framework, so you land exactly where you do your work. (If a brand‑new site hasn't created its `site/` folder yet, it falls back to the site root.)

If you haven't picked an editor, the app will prompt you to set one in Settings first.
